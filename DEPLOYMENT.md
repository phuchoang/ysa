## 🚀 Deployment Commands

### Web Deployment
```bash
# Export for web
npx expo export --platform web

# Deploy to EAS Hosting
eas deploy

# Production deployment
eas deploy --prod
```

### Mobile Builds
```bash
# Android build
eas build --platform android --profile production

# iOS build
eas build --platform ios --profile preview

# Check build status
eas build:list
```
